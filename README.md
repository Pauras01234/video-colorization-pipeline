# Video Colorizer

A project that turns black-and-white videos into color using open-source AI models.

## Goal
Build a clean, worthy project that:
- takes a black-and-white video
- extracts its frames
- colorizes each frame using an AI model
- rebuilds the frames into a final color video

## First model
DeOldify

## Planned pipeline
1. Input video
2. Extract frames
3. Colorize frames
4. Rebuild video
5. Save output
